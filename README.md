# Proposal Generator 🚀

A high-performance, web-based tool designed to generate professional, premium-branded `.docx` proposals for developers and agencies. This tool streamlines the proposal creation process, ensuring a consistent and state-of-the-art presentation for every client.

## ✨ Features

- **Premium Document Design**: Generates Word documents with native headers and footers, high-contrast layouts, and professional typography.
- **Platform Templates**: Includes hardcoded "Why Me" templates for **Shopify**, **WordPress**, and **Webflow** to quickly address common client objections.
- **Mobile Responsive**: A fully responsive UI that allows you to draft and manage proposals on the go with a dedicated mobile navigation menu.
- **Portfolio Management**: Add and edit portfolio projects with custom results, categories, and direct project links.
- **Dynamic Pricing**: Flexible phase-by-phase breakdown with automatic running totals and investment summaries.
- **Custom Branding**: Options to customize header subtitles, footer disclaimers, and personal stats.

## 🛠️ Tech Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3, and Modern JavaScript (ES6+).
- **Document Generation**: [docx.js](https://docx.js.org/) for programmatic `.docx` creation.
- **Typography**: Inter and Playfair Display (via Google Fonts).
- **Icons**: SVG-based iconography for a lightweight, crisp UI.

## 🚀 Getting Started

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/hamadrafi/Document-Generator.git
   ```
2. **Open the App**:
   Simply open `index.html` in any modern web browser (Chrome, Safari, Edge, or Firefox). No build step or local server is required.

## 📝 Usage Guide

1. **Step 1: Your Details**: Fill in your personal information, professional stats, and global branding preferences.
2. **Step 2-3: Project Context**: Define the client details and write your project understanding and personal intro.
3. **Step 4: Portfolio**: Add your best work. These will appear as professional "Project Cards" in the document.
4. **Step 5: Investment**: Define the project phases, pricing, and payment notes.
5. **Step 6: Why Me**: Select a platform template (Shopify/WP/Webflow) to auto-fill the comparison table, then tweak it to fit your expertise.
6. **Step 7: Final Steps**: Add your closing paragraph and call to action.
7. **Export**: Preview your data and click **"Download Proposal .docx"**.
